该方法一般用来合并多个obj类型对象,是传参时设置默认obj参数的一种方式.
主要有$.extend(a,b)$.extend({},a,b)$.extend(true,{},a,b)等几种方式,每种方式所产生的结果都不太一样.

  • $.extend(a,b,c…)
  • $.extend({},a,b,c…)
  • $.extend(true,a,b,c…)
    如果第一个参数为布尔值的话就开启深度拷贝模式,即obj中嵌套的obj也能互相拷贝而不是单纯的粗暴覆盖

同样的第一个obj{}不会改变target的值
